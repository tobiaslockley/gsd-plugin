CRITICAL: The cmd* functions call `core.output()` which writes to stdout and calls `process.exit(0)`.
The MCP server CANNOT call these directly — they would terminate the server process.

**Solution strategy:** The MCP server must intercept stdout and prevent process.exit:
Option A (recommended): Read the source of each cmd* function, extract the data-producing logic, and call the underlying fs/git operations directly.
Option B: Fork child processes via `execSync('node gsd-tools.cjs <command>')` and capture stdout — simpler but reintroduces the BashTool latency we are trying to eliminate.
Option C: Monkey-patch `process.exit` and capture `process.stdout.write` before calling cmd* functions — hacky but functional.

For this plan, use Option A where feasible (especially for simple operations like state reads). For complex operations where reimplementation is risky (like cmdPhaseComplete which updates multiple files), use Option B (child process) as a pragmatic compromise — it still eliminates the BashTool overhead since MCP tools invoke directly without prompt injection.

    **Implementation strategy for calling GSD lib functions:**

    Since cmd* functions call `process.exit()`, use a helper wrapper that:
    1. Captures stdout writes into a buffer
    2. Intercepts `process.exit()` to prevent server termination
    3. Returns the captured JSON output

    ```javascript
    function captureCmd(fn, ...args) {
      return new Promise((resolve, reject) => {
        const originalWrite = process.stdout.write;
        const originalExit = process.exit;
        let captured = '';

        process.stdout.write = (chunk) => { captured += chunk; return true; };
        process.exit = (code) => {
          process.stdout.write = originalWrite;
          process.exit = originalExit;
          if (code === 0 || code === undefined) {
            try { resolve(JSON.parse(captured)); } catch { resolve(captured); }
          } else {
            reject(new Error(`Command failed with code ${code}: ${captured}`));
          }
        };

        try {
          const result = fn(...args);
          // If fn returns without calling process.exit (some functions do)
          if (result instanceof Promise) {
            result.then(() => {
              process.stdout.write = originalWrite;
              process.exit = originalExit;
              try { resolve(JSON.parse(captured)); } catch { resolve(captured); }
            }).catch(err => {
              process.stdout.write = originalWrite;
              process.exit = originalExit;
              reject(err);
            });
          } else {
            // Synchronous function that called process.exit — handler above fires
            // If it returns normally without exit:
            process.stdout.write = originalWrite;
            process.exit = originalExit;
            try { resolve(JSON.parse(captured)); } catch { resolve(captured || result); }
          }
        } catch (err) {
          process.stdout.write = originalWrite;
          process.exit = originalExit;
          reject(err);
        }
      });
    }
    ```
